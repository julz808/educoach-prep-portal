## ‚ö†Ô∏è Budget Mistakes to Avoid

### ‚ùå **DON'T: Split budget equally**

**Wrong:**
```
All 6 campaigns get $16.67/day ($100 total √∑ 6)
```

**Why it's bad:**
- Not enough budget for winners to scale
- Wasting money on losers
- No data-driven decisions

### ‚ùå **DON'T: Set budget too low**

**Wrong:**
```
VIC Selective: $5/day
```

**Why it's bad:**
- Google needs ~15-20 clicks/day minimum to optimize
- At $1 CPC, $5 = only 5 clicks/day
- Not enough data for AI to learn

### ‚ùå **DON'T: Change budget daily**

**Wrong:**
```
Monday: $50/day
Tuesday: $20/day
Wednesday: $100/day
```

**Why it's bad:**
- Disrupts Google's learning
- Can't measure true performance
- Creates erratic results

### ‚úÖ **DO: Set weekly budgets**

**Right:**
```
Set budget on Monday
Don't touch it for 7 days
Review on Sunday
Adjust for next week
```
