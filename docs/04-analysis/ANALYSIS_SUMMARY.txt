ANIMATIONS & INTERACTIONS
================================================================================

Scroll-Based:
- Intersection Observer for fade-in animations
- Fade up, slide in from sides
- Staggered timing for grouped elements
- Triggered when element is 20% visible

Hover Effects:
- Scale (1.02 - 1.1)
- Shadow elevation
- Color transitions
- Rotation adjustments

Auto-Playing:
- Testimonials (5-second interval)
- School logos carousel (70-second cycle)
- Smooth linear transitions

Smooth Scrolling:
- Lenis smooth scroll library
- Lerp: 0.1
- Wheel multiplier: 0.8
- Smooth Touch: disabled

Spring Animations:
- Stiffness: 100-500
- Damping: 20-30
- Used for entrance animations

Duration Specs:
- Quick interactions: 300ms
- Standard transitions: 500ms
- Auto-play intervals: 5000ms (testimonials)
