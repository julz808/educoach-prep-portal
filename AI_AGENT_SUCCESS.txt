HOW TO APPROVE RECOMMENDATIONS:
================================================================================

Via Supabase (Current Method):
-------------------------------

1. Go to: https://supabase.com/dashboard
2. Select your project
3. Click "SQL Editor"
4. Run this query to see recommendations:

   SELECT
     id,
     action_type,
     details->>'action' as action,
     details->>'reasoning' as reasoning,
     details->>'expected_impact' as impact,
     details->>'priority' as priority,
     ai_confidence
   FROM google_ads_agent_actions
   WHERE requires_approval = true
     AND approved_at IS NULL
     AND rejected_at IS NULL
   ORDER BY
     CASE details->>'priority'
       WHEN 'high' THEN 1
       WHEN 'medium' THEN 2
       WHEN 'low' THEN 3
     END;

5. To APPROVE (replace XXX with the ID from query above):

   UPDATE google_ads_agent_actions
   SET
     approved_at = NOW(),
     approved_by = 'your-name'
   WHERE id = XXX;

6. To REJECT:

   UPDATE google_ads_agent_actions
   SET
     rejected_at = NOW(),
     rejected_by = 'your-name',
     rejection_reason = 'Too aggressive'
   WHERE id = XXX;

7. To approve ALL high priority recommendations:

   UPDATE google_ads_agent_actions
   SET
     approved_at = NOW(),
     approved_by = 'batch-approval'
   WHERE requires_approval = true
     AND approved_at IS NULL
     AND details->>'priority' = 'high';
