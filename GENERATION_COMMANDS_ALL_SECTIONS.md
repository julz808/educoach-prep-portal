Complete list of generation commands for all test products and sections.

- 1 diagnostic test
- ~180-360 questions per section (varies by section)

**Total Questions Generated (if all run):** ~7,000+ questions

---

## Usage Tips

1. **Run one at a time** - Monitor progress and costs
2. **Check output** - Each command shows questions generated and cost
3. **Re-run if needed** - Script automatically detects and fills gaps
4. **Check completion** - Look for "✅ All modes complete!" message

---

## After Generation

Once all sections are complete, generate drill questions:

```bash
npx tsx --env-file=.env scripts/generation/generate-drills-for-section.ts \
  --test="<TEST_TYPE>" \
  --section="<SECTION_NAME>" \
  --drills-per-subskill=20
```

Replace `<TEST_TYPE>` and `<SECTION_NAME>` with your values.
