# SEO Time Lag Strategy

## Core Principle: SEO Runs 10 Weeks Ahead of Google Ads

**Why?** Blog posts take 8-12 weeks to rank on page 1. Google Ads are instant.

## Key Insights

### 1. SEO Peaks Earlier Than Google Ads

**Google Ads PEAK:** 8 weeks before test (May 2026)
**SEO PEAK:** 17-13 weeks before test (Feb-March 2026)

### 2. Google Ads Peak = SEO Stop Writing

When Google Ads enters PEAK phase (8 weeks before test), SEO should **stop writing** because:
- Blog posts won't rank in time (need 8-10 weeks)
- Wasted effort - content will rank AFTER the test
- Better to focus on updating existing content

## Content Strategy by Phase

### TOO_EARLY (26+ weeks before test)
- **Google Ads:** Paused, wasting money
- **SEO:** TOO_EARLY, don't write yet
- **Posts/Week:** 0

### EARLY (16-26 weeks before test)
- **Google Ads:** 40% budget ($20/day)
- **SEO:** EARLY awareness content
- **Posts/Week:** 2 posts
- **Content Type:** Evergreen guides, "What is VIC Selective?", test format explanations

### RAMP_UP (11-16 weeks before test)
- **Google Ads:** 70% budget ($35/day)
- **SEO:** RAMP_UP content production
- **Posts/Week:** 3-4 posts
- **Content Type:** Study plans, sample questions, prep strategies

### PEAK (7-11 weeks before test)
- **Google Ads:** 100% budget ($50/day)
- **SEO:** PEAK content production (this is 17-13 weeks before test!)
- **Posts/Week:** 4-5 posts
- **Content Type:** Test-day checklists, last-minute tips, practice tests

### IMMINENT (1-7 weeks before test)
- **Google Ads:** 60% budget ($30/day)
- **SEO:** IMMINENT last-minute content (this is 11-7 weeks before test!)
- **Posts/Week:** 3 posts
- **Content Type:** "What to bring", "Day before checklist", "Common mistakes"

### POST_TEST (<10 weeks before test for SEO)
- **Google Ads:** Still running if >0 weeks
- **SEO:** STOP WRITING - content won't rank in time
- **Posts/Week:** 0
- **Action:** Update existing content instead of creating new

## Hard Rules for SEO Agent

### RULE 1: Enforce 10-Week Time Lag
```typescript
if (weeksUntilTest < 10) {
  return { posts: 0, reasoning: 'Content won't rank in time - STOP writing' };
}
```

### RULE 2: No Content in POST_TEST Phase
```typescript
if (seoPhase === 'POST_TEST') {
  throw new Error('Cannot write new content - test too close or finished');
}
```

### RULE 3: Update vs Create Decision
```typescript
if (weeksUntilTest >= 10 && weeksUntilTest <= 12) {
  // IMMINENT phase - prefer updates over new content
  return {
    new_posts: 1,
    content_updates: 3,
    reasoning: 'Focus on refreshing existing content that already ranks'
  };
}
```

## FAQ

### Q: Why 10 weeks specifically?

**A:** Industry data shows:
- Week 1-2: Google indexes post
- Week 3-4: Post appears on page 3-5
- Week 5-7: Post climbs to page 2
- Week 8-10: Post reaches page 1 (position 5-10)
- Week 12+: Post reaches position 1-3 (if high quality)

We use 10 weeks as the conservative estimate for page 1 ranking.

### Q: What if a post ranks faster than 10 weeks?

**A:** Great! That's a bonus. But we can't rely on it. The 10-week lag is a safety margin.

### Q: Should we ever write content <10 weeks before test?

**A:** Only if:
1. Updating existing content (already ranks)
2. Time-sensitive "day before" content (people search 1 day before test)
3. Post-test content ("What to do after the test")

But generally, NO - new content won't rank in time.

### Q: What about rolling test dates (ACER, EduTest)?

**A:** Use seasonal patterns instead:
- Q1 (Jan-Mar): PEAK writing (tests happen Q1)
- Q4 (Oct-Dec): RAMP_UP (10 weeks before Q1)
- Q2-Q3 (Apr-Sep): BASELINE (maintain existing content)

The 10-week lag still applies, just calculate from Q1 start date.

## Summary

**The 10-week time lag is CRITICAL for SEO success.**

Without it:
- ❌ Write content 2 weeks before test → ranks AFTER test → wasted effort
- ❌ Google Ads and SEO in sync → SEO always too late

With it:
- ✅ Write content 18 weeks before test → ranks 8 weeks before test → perfect timing
- ✅ SEO peaks when Google Ads is ramping → content ready when ads drive traffic
- ✅ Stop writing when Google Ads peaks → focus on conversions, not content

**Bottom line:** SEO is a 10-week leading indicator. Plan accordingly.
